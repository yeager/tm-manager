# TM Manager

Translation Memory (TMX) manager with fuzzy search.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install tm-manager
```

### Fedora/RPM
```bash
sudo dnf install tm-manager
```

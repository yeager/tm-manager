# Changelog

## [0.1.0] - 2026-02-14

### Added
- Initial release
- Open and create TMX files
- Fuzzy search in translation memory
- Import from PO, XLIFF, Qt TS formats
- Export to PO format
- Statistics view (segments, language pairs, projects)
- Filter by language
- Edit and delete individual segments
- Merge TMX files
- Modern GTK4/Adwaita interface
- Gettext i18n support
